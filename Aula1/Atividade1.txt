Elementos HTML são os blocos de construção fundamentais de uma página da web. Eles são as estruturas que compõem o conteúdo e a estrutura de uma página HTML.

5 - Pergunta: O que são elementos vazios em HTML? Forneça alguns exemplos de elementos vazios e explique por que eles são considerados vazios.

Elementos vazios em HTML, também conhecidos como elementos de autotag, são elementos que não possuem conteúdo entre uma tag de abertura e uma tag de fechamento em vez de usar uma tag de fechamento separada, esses elementos são fechados com uma barra ("/")

link rel="stylesheet" type="text/css" href="estilo.css"

6 - Pergunta: Explique a diferença entre elementos de bloco e elementos inline em HTML. Dê exemplos de cada um.

blocos: elementos de bloco são usados para estruturar o conteúdo da página em blocos distintos, eles normalmente começam em uma nova linha e ocupam toda a largura disponível do elemento pai que os contém.

inlines: elementos inline são usados para marcar partes de texto ou conteúdo que ocorrem dentro de elementos de bloco, eles não começam em uma nova linha e ocupam apenas o espaço necessário para o conteúdo que envolvem.

7 - Pergunta: Qual é a diferença entre as tags <div> e <span> em HTML? Quando você usaria uma em vez da outra?

div: é um elemento de bloco e é usado para criar um contêiner genérico que agrupa blocos de conteúdo

span:é um elemento inline e é usado para aplicar estilos a parte de um texto ou conteúdo dentro de um elemento de bloco

9 - Pergunta: Além do exemplo mencionado na aula, pesquise e liste pelo menos três meta informações que podem ser adicionadas a uma página HTML. Explique brevemente o propósito de cada uma delas.

Meta Charset (Meta de Conjunto de Caracteres): O atributo charset é usado para especificar o conjunto de caracteres usado no documento

Meta Description (Meta de Descrição):A meta descrição fornece uma breve descrição do conteúdo da página. Essa descrição é frequentemente usada pelos mecanismos de busca e é exibida nos resultados de pesquisa

Meta Keywords (Palavras-chave Meta):A meta keywords costumava ser usada para listar palavras-chave relevantes para o conteúdo da página.
